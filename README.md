# Tree Visualizer

A React Flow based Tree Visualizer project.

## Project Screenshot

![Tree Visualizer](tree-visualizer.png) 

## Features
- Interactive tree visualization
- Expand and collapse nodes
- Smooth edges between nodes
- Zoom and minimap support

## Technologies Used
- React
- React Flow
- Vite
